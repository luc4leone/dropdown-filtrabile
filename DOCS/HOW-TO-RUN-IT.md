0. genera i dati (dataset small ~2000, default):
   1) node proto-servizio/proto/scripts/build-generated-data.mjs
   2) node proto-servizio/proto/scripts/build-generated-data-small.mjs

1. seleziona il file `proto-servizio/proto/index.html`

2. in Windsurf pigia nella bottom status bar il bottone "Go Live"

nota: di default l'app usa `./generated/data.small.js`.
per usare il dataset grande: aggiungi `?big=1` all'URL.
